# Интерактивный учебник: тектоника христианского храма

Локальный статический мини-курс на русском языке про конструктивную логику храмовой архитектуры:
- колонны и пилоны;
- арки и распор;
- своды (цилиндрический/крестовый/ребристый);
- купол, барабан и пути передачи нагрузок.

## Запуск

Откройте `index.html` в браузере

или

```bash
python3 -m http.server 8000
# затем http://localhost:8000
```

## Что внутри

- Пошаговый маршрут обучения (6 модулей)
- Интерактивные схемы на `Canvas`
- Динамические контролы (ползунки/переключатели)
- Мини-квиз для самопроверки

Проект не требует сборки или установки зависимостей.
