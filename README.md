# Симулятор структурной механики христианской архитектуры (MVP)

Локальный браузерный симулятор, который объясняет конструктивную логику каменных церковных форм без тяжелой математики.

## Что внутри

- Vite + TypeScript + Three.js
- 5 интерактивных уроков MVP:
  1. Балка vs колонна
  2. Арка
  3. Свод
  4. Купол с парусами
  5. Контрфорс
- Качественная (не инженерно-точная) модель сил:
  - цветовые уровни риска (низкий/средний/высокий)
  - стрелки потоков усилий
  - зависимость от пролета, толщины, высоты, опирания

## Онтология (структурная)

Категории:
- **Силы**: сжатие, растяжение, изгиб, срез, распор, потеря устойчивости.
- **Элементы**: колонна, балка, стена, арка, свод, купол, парус, барабан, контрфорс.
- **Системы**: базиликальная, крестово-купольная.
- **Отказы**: трещины от растяжения, вываливание стены от распора, сдвиг опоры, buckling.

Примеры связей:
- Купол → генерирует распор → требует парусов/барабана/подпора.
- Арка → переводит вертикальную нагрузку в сжатие по дуге → создает горизонтальный распор.
- Свод → «серия арок» → длительный распор по стенам.
- Контрфорс → принимает распор → разгружает стену.

## Упрощенные правила симуляции

См. `src/sim/structuralRules.ts`.

Основные эвристики:
- Камень: высокая прочность на сжатие, низкая на растяжение.
- Риск балки растет со стройностью пролета (`span/depth`).
- Риск потери устойчивости колонны растет при `height/thickness`.
- Распор арки растет при большом пролете и малом подъеме.
- Для свода/купола опоры и массив подпорки уменьшают остаточный распор.

## Архитектура проекта

```text
src/
  content/ontology.ts       # онтология, правила, полная 10-урочная траектория
  lessons/index.ts          # 5 уроков MVP (геометрия + визуализация сил)
  sim/material.ts           # параметры «камня»
  sim/structuralRules.ts    # качественные структурные эвристики
  main.ts                   # UI, навигация уроков, 3D сцена
  style.css                 # layout + визуальный стиль
```

## Запуск локально

```bash
npm install
npm run dev
```

Открыть URL Vite (обычно `http://localhost:5173`).

## Педагогическая логика

Полная целевая траектория (10 уроков) включена в интерфейс справа:
1) путь нагрузки,
2) колонна/балка,
3) пределы каменной балки,
4) арка,
5) своды,
6) купола,
7) паруса,
8) контрфорсы,
9) базилика,
10) крестово-купольный тип.

MVP реализует центральные 5 уроков как минимально целостный прототип.
