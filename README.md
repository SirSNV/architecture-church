# Интерактивный учебник по тектонике христианского храма

Локальный учебник в формате статического сайта (HTML/CSS/JS) с интерактивными примерами:
- арки и распор,
- типы сводов,
- поведение купола,
- устойчивость колонн,
- мини-квиз на закрепление.

## Быстрый запуск локально

### Вариант 1: просто открыть файл
Откройте `index.html` в браузере.

### Вариант 2: через локальный сервер (рекомендуется)
```bash
python3 -m http.server 8000
```
Потом откройте `http://localhost:8000`.

## Технологии
- Чистые HTML/CSS/JS
- [Chart.js](https://www.chartjs.org/) (через CDN) для интерактивных графиков
- SVG для динамической схемы арки

## Педагогический дизайн
Учебник сделан по схеме:
1. Короткая теория
2. Манипуляция параметрами
3. Немедленная обратная связь
4. Проверка понимания через квиз

Математические модели в интерактивах упрощены и предназначены для формирования интуиции, а не для проектного расчёта конструкций.
